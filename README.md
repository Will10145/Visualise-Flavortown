## Visualise your Flavortown journey

![alt text](https://static.willcodes.tech/r/Flavortown_Visualiser.png)

How to use:

1. Head over to the [Live Demo](https://viaualiseflavortown.willcodes.tech/)
2. Get your projects URL, copy it
3. Paste the URL into the demo
4. Then get your API key by pressing the settings icon and scrolling down on flavortown, copy it, paste that as well
